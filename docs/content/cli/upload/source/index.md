# upload source

Manage upload sources for a space.

A source is a local filesystem path associated with a space. When you run `guppy upload`, Guppy scans these source paths to discover and upload files. Sources are local-only — they are not stored on the network.

## Usage

```
guppy upload source [command]
```

## Subcommands

### [add](add.md)

Add a source to a space.

### [list](list.md)

List sources added to a space.
